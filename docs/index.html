Использование LevelNet Endpoint Security App с установленным
антивирусным ПО, невосприимчивым к новой угрозе

1
Новая, ранее неизвестная угроза попадает на пользовательское устройство с установленным антивирусным ПО
2
Антивирусное ПО разрешает использование вредоносного объекта т.к. не считает его угрозой
3
LevelNet Endpoint Security App регистрирует факт проникновения неизвестного объекта и передает информацию о ней в P2P сеть
4
Описание угрозы регистрируется на серверах LevelNet для проверки на Falce Positive и защиту от атак злоумышленников
bug_reportsecurity
5
По результатам проверки принимается решение о блокировке угрозы и распостранении информации о ней внутри всей сети LevelNet за счет архитектуры P2P
6
Все пользовательские устройства
с установленным LevelNet Endpoint Security App защищены от ранее неизвестной угрозы

Использование LevelNet Endpoint Security App с установленным
антивирусным ПО, способным обнаружить угрозу

1
Известная угроза попадает на пользовательское устройство с установленным антивирусным ПО
2
Антивирусное ПО регистрирует факт появления известной угрозы
3
LevelNet Endpoint Security App фиксирует реакцию антивирусного ПО и передает информацию о ней в P2P сеть
4
Описание угрозы регистрируется на серверах LevelNet для проверки на False Positive и защиту от атак злоумышленников

5

6

с установленным LevelNet Endpoint Security App защищены от данного типа угрозы
